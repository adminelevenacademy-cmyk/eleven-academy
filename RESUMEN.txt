🚀 CÓMO EMPEZAR
═════════════════════════════════════════════════════════════════

PASO 1: Abre la app
  → Doble clic en: index.html
  → Se abre en Chrome automáticamente

PASO 2: Explora el Prospector
  → Busca academias de fútbol por país
  → Filtra por categoría (Academia, Colegio, Agente)
  → Busca por región

PASO 3: Agrega contactos al CRM
  → En cada contacto, haz clic "Agregar a CRM"
  → O crea leads manualmente en la sección CRM

PASO 4: Mueve leads por el pipeline
  → En CRM, arrastra entre etapas
  → O edita el lead y cambia estado

PASO 5: Envía emails
  → Ve a "Emails de Captación"
  → Elige un email (Día 0, Día 3, etc)
  → Copia el texto
  → Pega en Gmail y personaliza

PASO 6 (Opcional): Sincroniza con Mailchimp
  → Crea cuenta gratis: mailchimp.com
  → Ve a Configuración → Mailchimp API
  → Pega tu API Key, Server Prefix, Audience ID
  → Haz clic "Guardar y Probar Conexión"
  → Listo - desde ahora puedes sincronizar leads
