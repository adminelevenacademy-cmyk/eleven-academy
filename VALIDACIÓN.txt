✅ MAILCHIMP - PASOS DE CONFIGURACIÓN
═════════════════════════════════════════════════════════════════

PASO 1: Crear cuenta (gratis)
  > mailchimp.com
  > Sign up
  > No requiere tarjeta de crédito
  > Plan FREE: 500 contactos, 1.000 envios/mes

PASO 2: Obtener credenciales
  > Settings > Account & billing > Extras > API keys
  > Copiar: API Key (ej: xxxxxxxx-us21)
  > Server Prefix: Lo despues del guion (ej: us21)
  > Copiar Audience ID: Audience > Settings > Audience ID

PASO 3: Configurar en la app
  > Abrir index.html
  > Ir a: Configuración (abajo izquierda)
  > Sección: Mailchimp API
  > Pegar: API Key, Server Prefix, Audience ID
  > Click: "Guardar y Probar Conexión"

PASO 4: Usar
  > En CRM: Click "Sincronizar a Mailchimp"
  > Los leads se agregan automáticamente
  > Puedes enviar desde Mailchimp o desde la app

✅ RESOLUCION DE PROBLEMAS COMUNES
═════════════════════════════════════════════════════════════════

PROBLEMA: "No se abre la app"
SOLUCIÓN:
  1. Verifica que index.html esta en:
     C:\Users\Pirri\Desktop\CAPTACIÓN ELEVEN CLAUDE\index.html
  2. Doble clic en index.html
  3. Si abre editor de texto: Click derecho > Abrir con > Chrome

PROBLEMA: "Datos se borraron"
SOLUCIÓN:
  1. Los datos se guardan en localStorage
  2. Si limpias el navegador (Ctrl+Shift+Supr), se borran
  3. Siempre haz backup: Configuración > Exportar JSON
  4. Para restaurar: Configuración > Importar JSON

PROBLEMA: "Mailchimp no conecta"
SOLUCIÓN:
  1. Verifica que creaste cuenta en mailchimp.com
  2. Ve a Settings > Account & billing > Extras > API keys
  3. Copia exactamente tu API Key (todo, incluyendo guion y servidor)
  4. Server Prefix: Lo despues del ultimo guion (ej: "us21")
  5. Audience ID: Ve a Audience > Settings > Audience ID
  6. En la app: Configuración > Pega los datos
  7. Haz clic "Guardar y Probar Conexión"
  8. Si sigue sin funcionar: Escribe a mailchimp support

PROBLEMA: "No veo los contactos en el Prospector"
SOLUCIÓN:
  1. Recarga la pagina (F5)
  2. Los contactos estan pre-cargados
  3. Busca por pais: "Nigeria", "Ghana", "España", etc
  4. O busca por tipo: "Academia", "Colegio", "Agente"

PROBLEMA: "No puedo copiar emails"
SOLUCIÓN:
  1. Abre "Emails de Captación"
  2. Elige un email
  3. Llena las variables [Nombre], [Email], etc
  4. Haz clic "Copiar"
  5. Deberias ver mensaje verde "Copiado al portapapeles"
  6. Abre Gmail u Outlook
  7. Pega (Ctrl+V) en el cuerpo del email

PROBLEMA: "El navegador dice que esta lento"
SOLUCIÓN:
  1. La app esta optimizada
  2. Si esta lento: Close otras pestanas/apps
  3. Reinicia el navegador
  4. Si persiste: Borra cache (Configuración > Limpiar datos)
