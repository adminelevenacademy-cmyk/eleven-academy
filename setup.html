🚀 Bienvenido a Eleven Academy

App de captación de clientes - Guía de inicio

⚠️ La app YA ESTÁ LISTA. Este es un resumen rápido de lo que tienes.
1

App Web Funcional

Tienes una aplicación web completamente funcional con:

✅ Dashboard con métricas
✅ Prospector de 60+ contactos
✅ CRM con Kanban
✅ 10 emails pre-hechos
✅ Datos guardados localmente

2

Abre la app

Solo necesitas abrir el archivo index.html en Chrome:

C:\Users\Pirri\Desktop\CAPTACIÓN ELEVEN CLAUDE\index.html

Doble clic → Chrome abre automáticamente

3

Configurar Mailchimp (Opcional pero recomendado)

Para enviar emails automáticos:

1. Crea cuenta gratis: mailchimp.com
2. Obtén tu API Key (Settings → API keys)
3. Obtén tu Audience ID
4. En la app: Configuración → Mailchimp API → Pega datos

4

Comienza a prospectar

Acciones principales:

• Prospector: Busca contactos por país/categoría
• Agrega a CRM: Haz clic en contacto → "Agregar a CRM"
• Gestiona: Mueve leads en el pipeline
• Emails: Copia los 10 emails listos y envía
• Sincroniza: Opcional - envía a Mailchimp

✓ Todo funciona sin instalar nada ni escribir código.
