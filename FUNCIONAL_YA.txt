🔧 SETUP MAILCHIMP (5 MINUTOS)
═════════════════════════════════════════════════════════════════

PASO 1: Crear cuenta (GRATIS)
   1. Ve a: https://mailchimp.com
   2. Haz clic: "Sign Up"
   3. Llena tu email
   4. Crea contraseña
   5. NO requiere tarjeta de crédito
   6. Confirma tu email

PASO 2: Obtener API KEY
   1. Entra a tu cuenta Mailchimp
   2. Click en tu usuario (arriba derecha)
   3. Ve a: Settings → Account & billing
   4. Click en: Extras → API keys
   5. Bajo "Your API keys", copia la primera API Key
      (Ejemplo: xxxxxxxx-us21)

PASO 3: Obtener SERVER PREFIX
   1. La API Key tiene formato: xxxxxxxx-us21
   2. El SERVER PREFIX es lo después del guion
   3. En el ejemplo anterior: us21

PASO 4: Obtener AUDIENCE ID (List ID)
   1. En Mailchimp, click en: Audience
   2. Click en: Settings
   3. Click en: Audience name and campaign defaults
   4. Copia el "Audience ID" (largo código)

PASO 5: Configurar en la app
   1. En index.html (ya abierto):
   2. Ve a: Configuración (abajo izquierda)
   3. Sección: "Mailchimp API"
   4. Pega:
      - API Key
      - Server Prefix
      - Audience / List ID
   5. Click: "Guardar y Probar Conexión"
   6. Si ves ✓ verde = ¡FUNCIONA!
