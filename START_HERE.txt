4️⃣ MAILCHIMP GRATIS (Opcional):

   No necesitas nada complejo. Dos opciones:

   OPCIÓN A: Manual (Sin código)
   └─ Crea cuenta: mailchimp.com (gratis)
   └─ Ve a Audience → Contacts
   └─ Haz clic "Import"
   └─ Sube un CSV con tus contactos
   └─ Listo - puedes enviar desde Mailchimp

   OPCIÓN B: Integrado en la app
   └─ En "Configuración" → Mailchimp API
   └─ Pega tu API Key (Settings → API keys)
   └─ Pega Server Prefix (está en tu API Key)
   └─ Pega Audience ID (Audience → Settings)
   └─ Haz clic "Guardar y Probar Conexión"
   └─ ¡Listo! La app sincroniza automáticamente

🆘 PROBLEMAS COMUNES:

   P: "No se abre la app"
   R: Asegúrate que index.html está en la carpeta correcta:
      C:\Users\Pirri\Desktop\CAPTACIÓN ELEVEN CLAUDE\index.html

   P: "Datos se perdieron"
   R: Los datos están en localStorage (permanente en tu PC)
      Salvo que limpies el navegador
      Siempre haz backup: Configuración → Exportar JSON

   P: "Mailchimp no conecta"
   R: 1. Verifica que creaste cuenta en mailchimp.com
      2. Ve a Settings → Account & billing → Extras → API keys
      3. Copia correctamente API Key, Server Prefix, Audience ID
      4. Pega en la app y haz clic "Guardar y Probar Conexión"

   P: "Quiero que los datos se sincronicen automáticamente"
   R: Opciones:
      - Instala Node.js (gratis) + ejecuta: npm install && npm start
      - O usa Zapier (free/PRO) para automatizar
      - O contratar desarrollador para backend completo
